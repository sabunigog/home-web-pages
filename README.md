# home-web-pages

本仓库用于归档和发布多个独立的 HTML 页面。这些页面可以作为静态网页直接访问，适用于存放个人项目、工具页面、备忘页面或演示页面。

## 项目简介

`home-web-pages` 是一个简单的 HTML 页面集合仓库。每个页面都是独立的，不依赖于特定的域名或后端服务。你可以通过 GitHub Pages 或其他静态托管服务直接浏览这些页面。

## 目录说明

```
.
├── index.html          # 仓库首页（可选）
├── page1/              # 页面目录示例
│   └── index.html
├── page2/
│   └── index.html
└── README.md           # 本文件
```

- 每个页面建议放在独立的子目录中，以 `index.html` 命名，便于直接访问。
- 根目录下的 `index.html` 可作为页面导航或仓库简介。

## 页面归档说明

- 所有页面均为纯 HTML，支持嵌入 CSS 和 JavaScript。
- 页面之间无依赖关系，可独立运行。
- 如需添加新页面，请创建新目录并放入 `index.html` 文件。

## 维护说明

- 欢迎提交 Pull Request 添加新的页面或改进现有页面。
- 请保持页面内容简洁，避免引入外部资源导致加载缓慢。
- 不建议在页面中包含广告、跟踪脚本或商业推广内容。
- 本仓库仅用于归档和展示，不提供任何形式的服务保证。

## 使用方式

1. 克隆本仓库到本地。
2. 在 `docs/` 或根目录下添加你的 HTML 页面。
3. 推送到 GitHub 后，可通过仓库的 Pages 功能访问。
